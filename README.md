## 규칙
각 조원은 본인의 폴더에 과제를 제출합니다.

각 조원들은 본인의 브랜치를 따로 생성한 후 develop에 pull request를 올립니다.

제출된 과제에 대하여 각 조원들은 코드 리뷰를 할 수 있습니다. 리뷰할 때, 조원의 코드로 배운점, 본인이 제시하고 싶은 코드의 방식, 코드를 확인하며 궁금한 점 등 다양하게 리뷰할 수 있습니다.
